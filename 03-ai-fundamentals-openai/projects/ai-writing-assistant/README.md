# AI Writing Assistant

## Features
- Streaming responses
- Conversation history
- Multiple writing modes (creative, technical, simple)
- Cost tracking
- Token management

## Usage
```bash
python assistant.py
```

## Commands
- `/mode <creative|technical|simple>` - Change writing style
- `/stats` - Show usage statistics
- `/clear` - Clear conversation
- `quit` - Exit

See `assistant.py` for complete implementation combining all Phase 3 concepts.
