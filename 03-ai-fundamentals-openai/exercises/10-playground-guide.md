# OpenAI Playground Guide

## What is the Playground?
Web interface for testing OpenAI models without code.

## Key Features
1. **Model selection** - Try different models
2. **Parameter tuning** - Adjust temperature, tokens, etc.
3. **System prompt testing** - Test prompts quickly
4. **Share presets** - Save and share configurations

## Exercises
1. Test same prompt with temperature 0 vs 2
2. Compare GPT-3.5 vs GPT-4 on complex task
3. Save 3 useful system prompt presets
4. Experiment with max_tokens limits
5. Test streaming vs non-streaming
